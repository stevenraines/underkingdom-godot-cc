## Controls / Keybindings

Grouped by screen/context. Many screens also accept the arrow keys for navigation and `Enter`/`Space` to confirm.

- **Global / Map**
	- Movement: Arrow keys or W/A/S/D
	- Wait / Rest: `.` (period). Hold to repeat.
	- Descend stairs / Enter dungeon: `Shift` + `.` (or `>`)
	- Ascend stairs: `Shift` + `,` (or `<`)
	- Interact / Use structure: `E`
	- Talk / interact with NPC: `T`
	- Toggle Inventory: `I`
	- Open Crafting: `C`
	- Toggle Build Mode: `B`
	- Toggle World Map: `M`
	- Toggle Auto-pickup: `G`
	- Manual Pickup: `,` (comma)
	- Harvest mode (choose direction): `H`
	- Restart when dead: `R`
	- Cancel / Close / Back: `Esc`

- **Inventory Screen** (open with `I`)
	- Close: `Esc` or `I`
	- Navigate items: `Up` / `Down`
	- Switch panel (Equipment / Backpack): `Tab`
	- Equip / Unequip: `E`
	- Use (consume): `U`
	- Drop: `D`
	- Action / Select: `Enter` / `Space`
	- Quick slot number shortcuts when equipping: `1`-`9`
	- Slot picker: `1`-`9` quick-select, `Enter` to confirm, `Esc` to cancel

- **Crafting Screen** (open with `C`)
	- Close: `Esc` / `ui_cancel`
	- Navigate recipes: `Up` / `Down`
	- Craft / Confirm: `Enter` / `ui_accept`
	- UI blocks player movement while open

- **Build Mode**
	- Toggle / Close: `B` or `Esc`
	- Navigate build cursor / list: `Up` / `Down` (and arrow keys)
	- Place structure: `Enter` / `Space`
	- Quick-select structure: `1`, `2`, `3`, ...

- **Container Screen**
	- Close: `Esc` or `E`
	- Navigate: `Up` / `Down`
	- Switch panels: `Tab`
	- Take / Put / Confirm: `Enter` / `Space`

- **Shop Screen**
	- Close: `Esc`
	- Navigate items: `Up` / `Down`
	- Switch panel (buy/sell): `Tab`
	- Buy / Sell / Confirm: `Enter`
	- Toggle sell/buy mode: `S`
	- Increase quantity: `+` (`=`) / keypad `+`
	- Decrease quantity: `-` / keypad `-`

- **World Map Screen** (open with `M`)
	- Close: `Esc` or `M`
	- Shows overview of the island with biome colors
	- Displays player position, town, and all dungeon entrances
	- Legend shows what each marker represents

## Recent Features

### Multi-Dungeon System
The world now spawns multiple dungeon types, each with unique characteristics:
- **Burial Barrow** - Ancient tomb with undead (grassland/woodland)
- **Natural Cave** - Organic caverns (mountains/rocky hills)
- **Abandoned Mine** - Mining tunnels with shafts (mountains)
- **Sewers** - Underground waste system (town area)
- **Temple Ruins** - Forgotten sacred site (forest/rainforest)
- **Ancient Fort** - Ruined fortress (grassland/hills)
- **Military Compound** - Organized defenses (grassland/woodland)
- **Wizard Tower** - Arcane dangers (anywhere)

Each dungeon entrance has a unique symbol and color. Stand on an entrance and press `>` to enter.

### Destroyable Features
Some dungeon features (like rat nests) can be permanently destroyed when interacted with.
