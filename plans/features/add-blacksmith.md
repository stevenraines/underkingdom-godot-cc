# Feature: Add Blacksmith

**Goal**: Add a blacksmith who makes armour and weapons and can train the player in crafting

---

The starter town should have a blacksmith shop with an NPC blacksmith. The character should trade items that are armor or weapons. The npc can also train the player to craft armor and weapons. Any receipe with metal requires a forge and an anvil. Make those as structures like the campfire - as things the player may be near for the puposes of crafting.

To use the anvil, the player MUST have a hammer. It does not need to be equipped. If the player does not, show a message indicating it is required.

To use the forge, the player MUST have tongs. It does not need to be equipped. If the player does not, show a message indicating it is required.
