# Feature - inscribe / uninscribe objects
**Goal**: Player can add (or remove) inscriptions to objects.

---
Implement a system to support inscribing/uninscribing objects.

Inscribe an object ({)
This command inscribes a string on an object. The inscription is displayed inside curly braces after the object description. The inscription is limited to the particular object (or stack) and is not automatically transferred to all similar objects

Uninscribe an object (})
This command removes the inscription on an object.
